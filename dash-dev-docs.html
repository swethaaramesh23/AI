Quick Start Guide

To authenticate your API requests, include your API key in the Authorization header as a Bearer token.

curl -X POST https://api.stackly.ai/v1/chat \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello"}'

Available Endpoints

  • /v1/chat - AI conversational endpoints
  • /v1/models - Fine-tuning and model management
  • /v1/billing - Usage and invoice retrieval
